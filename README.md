********** INTRUCOES PARA RODAR MAIN.EXE **********
 -> Necessario ter o programa MinGW e compilador gcc instalado no seu Windows
 -> Apos conferir que esta tudo instalado, basta executar o arquivo main.exe localizado dentro da pasta "LocProg---Sistema-de-loca-o-de-veiculo-main\LocProg---Sistema-de-loca-o-de-veiculo-main\build"
 -> Caso deseje alterar a disponibilidade de qualquer veiculo, basta entrar na pastav "LocProg---Sistema-de-loca-o-de-veiculo-main\LocProg---Sistema-de-loca-o-de-veiculo-main\data"
   -> e alterar o status para 0 - INDISPONIVEL // 1 - DISPONIVEL
 -> Para adicionar qualquer veiculo basta seguir o padrao e adicionar antes do /0 na ultima linha
   -> seguindo o padrao de (MARCA - MODELO - COR - VALOR DA DIARIA - DISPONIBILIDADE (0 ou 1) )
